# UTM Tracking System

Система для отслеживания UTM меток и аналитики переходов из TikTok/Instagram в Telegram.

## Быстрый старт

1. Создайте UTM ссылку через API
2. Используйте короткую ссылку для рекламы
3. Отслеживайте клики и конверсии

## Текущий статус

- ✅ API работает на Railway
- ✅ База данных PostgreSQL
- ✅ Redis для кеширования
- ✅ Cloudflare Worker для коротких ссылок
- ✅ Landing pages с авто-редиректом
- ✅ Отслеживание кликов
- ⏳ Интеграция с Telegram ботом (в процессе)
